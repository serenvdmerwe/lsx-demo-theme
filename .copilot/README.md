# LSX Demo Theme - Copilot Configuration

This directory contains configuration files for GitHub Copilot to enhance its understanding of the LSX Demo Theme project.

## Files in this Directory

- `mcp.json` - Model Context Protocol configuration for Copilot
- `configuration.md` - Detailed guidance for Copilot when working with this theme

## Purpose

These configuration files help GitHub Copilot:

1. Understand the structure of the LSX Demo Theme
2. Follow WordPress and theme-specific coding standards
3. Generate code that matches the project's patterns and conventions
4. Provide more contextually relevant suggestions when working with theme files

## How to Use

You don't need to manually interact with these files. GitHub Copilot will automatically use them to provide better context-aware assistance when you're working in this repository.

If you need to update the Copilot configuration:

1. Edit the appropriate file in this directory
2. Commit and push your changes
3. Other team members will automatically benefit from the updated configuration
